#YouTube Albums

## Installation
1. `npm install`


## Build for dev
1. `npm run build-dev`
2. go to `localhost:2222`

## Build for prod 
1. `npm run build-prod`
2. app is built in `./dist` folder, you can manually serve the app with any http-server
   
    e.g
    - `npm instlal http-server -g`
    - `http-server ./dist`
